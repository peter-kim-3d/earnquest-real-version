Creating Custom Tasks

  1. Navigate to "Manage Tasks" in your dashboard
  2. Click "Add New Task"
  3. Fill in task details (name, category, points)
  4. Choose approval type (manual, automatic, or timer-based)
  5. Assign to specific children or all
  6. Save and activate

üìù Best Practices for Tasks

  • Keep tasks specific and measurable
  • Start with easier tasks to build momentum
  • Adjust point values based on task difficulty
  • Use photos for verification when needed
  • Review and adjust weekly based on completion rates

Reviewing Task Submissions

Approval Process

When a child completes a task requiring approval, you'll receive a notification. Here's how to review:

  1. Open the Pending Approvals section
  2. Review the task details and any submitted evidence (photos, notes)
  3. Choose to Approve, Reject, or Request Changes
  4. Add optional feedback or encouragement

‚ö†Ô∏è Important: Timely Approvals

Try to review tasks within 24 hours. Quick feedback keeps children motivated and engaged. If you can't review immediately, enable auto-approval for trusted tasks.

EarnQuest comes with pre-configured task and reward sets for different age groups:

  • Mix of simple and complex tasks

  • Focus on habit formation

Ages 12-14: Learning Focus

  • More complex, self-directed tasks
  • Higher point values (30-100 points)
  • Emphasis on academic and life skills
  • Greater independence and trust
  • Goal-setting and planning skills

Screen Time Management

One of EarnQuest's most popular features is integrated screen time control.

How It Works

  1. Set up device screen time limits in your device settings
  2. Connect EarnQuest to these controls (optional)
  3. Children earn "screen time tickets" by completing tasks
  4. Each ticket unlocks a specific amount of device time
  5. System automatically manages permissions

üéÆ Why This Works

Instead of nagging or conflict over screen time, the system is clear and fair. Kids understand exactly how to earn their device time, and parents don't have to be the "bad guy."

Setup Instructions

  1. Go to Settings ‚Üí Screen Time Integration
  2. Enable screen time management
  3. Set daily screen time limits (recommended: 1-2 hours)
  4. Choose point cost per 30 minutes (recommended: 50-100 points)
  5. Connect child's device (requires device permission)

Approval Types

EarnQuest offers multiple ways to verify task completion:

1. Manual Approval

Parent manually reviews and approves. Best for:

  • Tasks requiring quality check (room cleaning, homework)
  • New tasks or new children
  • High-value tasks

2. Automatic Approval

Task is approved immediately upon submission. Best for:

  • Trusted children
  • Simple, verifiable tasks
  • Tasks done in parent's presence

3. Timer-Based

Child sets a timer and task approves when time is complete. Best for:

  • Reading time
  • Exercise duration
  • Study sessions

4. Photo Verification

Child submits photo proof with task. Best for:

  • Chores (clean room, organized desk)
  • Creative projects
  • Before/after tasks

5. Checklist

Child checks off multiple steps. Best for:

  • Multi-step homework
  • Morning routines
  • Bedtime routines

Frequently Asked Questions

Common questions and answers

What age is EarnQuest suitable for? +
EarnQuest is designed for children ages 5-14, with specific features and presets for different age groups. Younger children (5-7) need more parent involvement, while older children (12-14) can operate more independently.
How many children can I add to one family account? +
You can add unlimited children to your family account. Each child gets their own dashboard, tasks, and rewards. The system is designed to handle multiple children fairly.
Can both parents manage the account? +
Yes! You can invite your partner to be a co-parent on the account. Both parents can approve tasks, manage rewards, and adjust settings. All changes sync in real-time.
What if my child completes a task but I reject it? +
If you reject a task, you can provide feedback on why it wasn't acceptable. The child can then redo the task and resubmit. This teaches quality standards while still being encouraging. Try to reject only when necessary and always explain clearly.
Should I use real money as rewards? +
This is a personal choice. Some families include small monetary rewards (like $1 = 100 points), while others stick to experiences and privileges. Consider your family values and financial situation. Many experts recommend emphasizing non-monetary rewards for younger children.
How do I prevent my child from "gaming the system"? +
The trust level system helps with this. If a child is dishonest, they lose trust levels and require more supervision. Also, use photo verification and spot checks for high-value tasks. Most importantly, talk about honesty and integrity - the system works best when everyone plays fair.
What's the difference between regular tasks and kindness recognition? +
Regular tasks are expected responsibilities that earn points. Kindness recognition celebrates unexpected, prosocial behavior that shouldn't be "paid for" - like spontaneously helping a sibling or showing empathy. These earn special acknowledgments but not points, teaching intrinsic motivation.
Can I use EarnQuest offline? +
Basic features work offline - children can view tasks and mark them complete. However, syncing, approvals, and store purchases require an internet connection. Any offline actions will sync when connection is restored.
What if my child loses interest? +
Try refreshing the rewards with something exciting, adjusting task difficulty, or taking a short break and reintroducing it. Also ensure you're celebrating successes and keeping the experience positive. Some children need time to adjust to the system.
Is there a cost to use EarnQuest? +
EarnQuest offers a free tier with basic features for one child. Premium plans unlock multiple children, advanced features like screen time integration, custom branding, and priority support. Check the pricing page for current options.
